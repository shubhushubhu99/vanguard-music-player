Installation & System Requirements
Currently, Vanguard Player is fully optimized and compiled for Windows (.exe) and Linux (.AppImage).

Privacy & Technical
No. Vanguard Player is built for absolute privacy. There is zero telemetry, zero tracking, and zero ads. Your data, playlists, and listening habits stay entirely locally on your own machine.
Instead of using heavy web wrappers like Electron, Vanguard Player is built using Tauri and Rust. This allows the app to interact directly with your operating system, keeping RAM and CPU usage incredibly low while maintaining a strikingly sleek React UI.
Under the hood, it utilizes yt-dlp for lightning-fast, secure audio scraping, and mpv as the highly versatile background audio engine, communicating seamlessly via Unix IPC sockets.
